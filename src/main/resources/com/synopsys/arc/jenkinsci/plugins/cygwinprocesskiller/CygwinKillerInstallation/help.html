This installation defines a CYGWIN_HOME directory, which should be used for termination of Cygwin processes. In order to achieve proper behavior of default termination script, entire cygwin process tree should the same cygwin1.dll
Cygwin installation should contain following utilities:
  • interpreter defined in the configuration
  • uname for checking of Unix architecture
  • all shell commands invoked from the termination script
  • all dependencies from the list
See the plugin&squot;s page for more info and requirements.
